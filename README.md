## 设置与运行

1. 克隆仓库:
```
git clone <repository-url>
cd donkey-website
```

2. 安装依赖（如果需要）:
```
npm install
```

3. 本地运行:
```
# 如果使用Node.js服务器:
npm start

# 或者使用简单的HTTP服务器:
python -m http.server
```

4. 打开浏览器访问 `http://localhost:8000` 或相应端口。
